=== Rich Related Posts ===
Version: 1.0.0
Tags: related, sidebar, posts, shortcode
Requires at least: 2.7
Tested up to: 2.8.2

The purpose of the plugin is to facilitate related posts functionality where posts can be enriched with groups of links to blog posts (or other sites), defined by headline and thumbnail.

== Description ==

Powerful CMS solution to quickly and easily create multiple groups of rich related content with images using tools such as combination search/entry fields and element units consisting of custom headline + permalink + image url. Code is ready to be inserted into posts, and can be used in sidebar text widgets after adding this filter to functions.php:

add_filter('widget_text', 'do_shortcode');  

== Installation ==
* Upload folder `rich-related-posts` to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
